# Zumito Documentation

This repository hosts the public documentation site for the **Zumito Framework**. It is built with [Astro](https://astro.build/) using the Starlight theme.

## Install dependencies

```bash
npm install
```

## Build locally

```bash
npm run build
```

The generated site will be available in the `dist` folder.

## Live docs

Visit [docs.zumito.dev](https://docs.zumito.dev) for the latest published documentation.
