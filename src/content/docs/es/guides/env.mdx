Aquí puedes encontrar todos los parámetros que se pueden definir en archivos `.env` o variables de entorno. Normalmente establecemos aquí variables que pueden diferir entre ambientes (producción, desarrollo, CI, etc.) como tokens o dominios de APIs.

Puedes usar el archivo `.env.example` como punto de partida para tu configuración.

## Discord variables

| Nombre | Descripción | Ejemplo |
| ------ | ----------- | ------- |
| DISCORD_TOKEN | Token del bot de Discord | `1234567890abcdef` |
| DISCORD_CLIENT_ID | ID pública del bot | `123456789012345678` |
| DISCORD_CLIENT_SECRET | Clave secreta privada del bot | `abcdefghijklmnopqrstuvwxyz123456` |

## Database variable

| Nombre | Descripción | Ejemplo |
| ------ | ----------- | ------- |
| MONGO_URI | Cadena de conexión de MongoDB | `mongodb://usuario:clave@localhost:27017/basedatos` |

## Other variables

| Nombre | Descripción | Ejemplo |
| ------ | ----------- | ------- |
| BOT_PREFIX | Prefijo para comandos por prefijo. Se usará como prefijo por defecto/alternativo. | `!` |
| SECRET_KEY | Se utilizará para cifrar datos como los tokens JWT en la API. | `supersecreto` |
