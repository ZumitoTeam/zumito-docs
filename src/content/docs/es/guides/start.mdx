Esta guía le ayudará a comenzar rápidamente a desarrollar un nuevo bot utilizando zumito-framework.

## Requisitos previos

- [Node.js](https://nodejs.org/en/) (v18 o superior)
- [Git](https://git-scm.com/)


- Discord Bot **Token**, puedes conseguir uno [aquí](https://discord.com/developers/applications)
- Discord Bot **Client ID**, puedes conseguir uno [aquí](https://discord.com/developers/applications)
- Discord Bot **Client Secret**, puedes conseguir uno [aquí](https://discord.com/developers/applications)

## Setup

<Steps>
  1. Crear un nuevo proyecto
      <Tabs>
        <TabItem label="Command Line" icon="cli">
      ```bash
      npx zumito-cli create project
      ```
        </TabItem>
        <TabItem label="Wizard" icon="wizard">
          <CreateProject client:only="svelte"/>
        </TabItem>
      </Tabs>

  2. Inicia el bot
      ```bash
      npm run dev
      ```
</Steps>

Se clonará el proyecto de plantilla, se instalarán las dependencias y se configurará el archivo `.env` mediante algunas preguntas.

La configuración de la base de datos se guardará en el archivo `.env`. Puedes cambiarla más tarde si lo deseas.
- **Tingodb** se recomienda para el **desarrollo**, ya que no requiere una base de datos externa y guardará los datos en un archivo.
- **MongoDB** se recomienda para **producción** debido a su mejor rendimiento.

<Aside>Puedes encontrar más información sobre el archivo `.env` [aquí](/docs/bot/env).</Aside>
