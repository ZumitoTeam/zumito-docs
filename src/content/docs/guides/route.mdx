This guide will help you to quickly start developing a new bot using zumito-framework.

## What's a route?

A route is an a web endpoint to allow users or other programas to interact and retrieve information about your bot.

## How can i create a route?

<Steps>
  1. Create new route
      <Tabs>
        <TabItem label="Command Line" icon="cli">
      ```bash
      npx zumito-cli create route
      ```
        </TabItem>
        <TabItem label="Wizard" icon="wizard">
          Work in progress...
        </TabItem>
      </Tabs>

  2. Edit your file
      :::note
        You will find the new route file in `/src/modules/<YourModule>/routes`
      :::
      You will find a method like this:
      ```ts
      execute(req, res) {
        
      }
      ```
      Here you can run your route code. req and res parameters match the [Express server params](https://expressjs.com/en/guide/routing.html)

</Steps>

## Quick tips

:::tip[Redirects]
    you can send redirect with: 
    ```ts
    res.redirect(`https://framework.zumito.dev`);
    ```
:::
